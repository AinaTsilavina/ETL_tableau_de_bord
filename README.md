# 🔄 ETL Data Processing System

## 🧩 Description

Ce projet implémente un processus ETL (Extract, Transform, Load) permettant de collecter, nettoyer, transformer et stocker des données pour leur exploitation.

L’objectif est d’assurer la qualité, la cohérence et la disponibilité des données pour les applications analytiques.

---

## 🚀 Fonctionnalités

- 📥 Extraction de données depuis différentes sources
- 🧹 Nettoyage et transformation des données
- 🔄 Automatisation du pipeline ETL
- 📊 Structuration et chargement en base de données
- ⚡ Optimisation des performances de traitement

---

## 🏗️ Architecture

Le pipeline ETL est structuré en trois étapes :

1. **Extract** : récupération des données
2. **Transform** : nettoyage et transformation avec Pandas
3. **Load** : insertion dans la base de données

---

## 🛠️ Technologies utilisées

- Python
- Pandas
- SQL (SQLite / MySQL)
- Django
- Git
